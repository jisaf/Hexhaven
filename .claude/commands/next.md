# Bug Report and Diagnosis

Investigate and diagnose the reported bug. Follow these steps:

1. Gather information about the bug (symptoms, error messages, reproduction steps)
2. Search the codebase for relevant code
3. Identify the root cause
4. Propose a fix
5. If requested, implement the fix with tests

Ask clarifying questions if needed to fully understand the issue.
